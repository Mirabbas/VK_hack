Описание объекта сообщения¶

class Message¶
answer(msg: str, **additional_values)¶

(Сопрограмма) Функция, посылающая сообщение msg отправителю сообщения с дополнительными параметрами additional_values. Стоит использовать в случае, когда это последнее или единственное сообщение от плагина.

send(msg: str, **additional_values)¶

(Сопрограмма) Функция, посылающая сообщение msg отправителю сообщения с дополнительными параметрами additional_values.

vk¶

экземпляр VkPlus, объект, через который бот взаимодействует с ВКонтакте.

conf¶

True, если сообщение пришло из беседы, иначе False

user¶

Экземпляр User из базы данных

cid¶

Id беседы или None, если сообщене не из беседы

user_id¶

Id автора сообщения

peer_id¶

Отправитель (id беседы, пользователя без разбора, лучше использовать user_id и cid)

text¶

Сообщение пользователя без префикса и команды

body¶

Сообщение пользователя

prefix¶

True, если сообщение содержит префикс, иначе False

is_out¶

True, если сообщение исходящее, иначе False

timestamp¶

UNIX-время отправки сообщения

brief_attaches¶

Индикатор вложений сообщения, содержит вложения, но не гарантировано, что все и не гарантировано, что они будут содержать access_key

full_attaches¶

(Сопрограмма, атрибут) которая возвращает все вложения

brief_forwarded¶

Список с ID пересылаемыми сообщениями формата (id сообщения, id пересылаемых сообщении).

full_forwarded¶

(Сопрограмма, атрибут) Список с ID пересылаемыми сообщениями формата (id сообщения, id пересылаемых сообщении).

msg_id¶

Id сообщения

command¶

Команда в сообщении, если присутствует
