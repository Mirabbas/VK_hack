Описание объекта VkPlus¶

class VkPlus¶
method(method: str, **values, send_from=None, nowait=False)¶

(Сопрограмма) Выполняет метод method API ВКонтакте (например, «messages.get») с параметрами values (например, {«count»: 2}), в send_from можно передать SenderGroup или SenderUser, чтобы уточнить, кто должен выполнять команду, если nowait=True, то method не вернёт результат запроса к вк, а продолжит пограмму без ожидания ответа.

upload_doc(multipart_data, filename="image.png")¶

(Сопрограмма) Загружает multipart_data с именем filename и возвращает Attachment.

upload_photo(multipart_data)¶

(Сопрограмма) Загружает фото multipart_data и возвращает Attachment.

mark_as_read(ids: str)¶

(Сопрограмма) Помечает сообщения с ids как прочитанные.

resolve_name(screen_name: str)¶

(Сопрограмма) Перевод короткого имени screen_name в числовой ID.

get_default_sender(key)¶

Возвращает подходящего (на сколько это возможно) отправителя для метода key.

Attachment(token_id):
type¶

Тип вложения (photo, video, audio, doc, link, market, market_album, wall, wall_reply, sticker или gift).

owner_id¶

ID владельца вложения.

id¶

ID вложения.

access_key¶

access_key вложения, если присутствует.

url¶

Сслыка на вложение, если присутствует.

SenderGroup(token_id):

Какой токен должен выполнять запрос, token_id - это порядковый номер в текущем списке групповых личностей бота.

SenderUser(token_id):

Какой пользователь должен выполнять запрос, user_id - это порядковый номер в текущем списке пользовательских личностей бота.
